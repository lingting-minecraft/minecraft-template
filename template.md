#### 项目说明

- 包含一个示例的物品和方块
- 包含方块实体示例
- 包含方块状态切换以及对应渲染切换示例
- 包含国际化示例
- 包含掉落物自定义示例
- 自定义合成表示例
- 支持多个创建模式物品栏页签(CreativeTabs)
- 自定义悬浮文本示例
- 方块掉落物支持存储nbt和state
- 方块限制工具示例
- 方块区分是否精准采集示例
- 支持自定义命令
- 支持矿石方块自然生成 [命令行调试文档](./help_command.md#place)
- 包含台阶示例

#### 资源

- 统一放在 assets/ 文件夹下

| 名称       | 说明   |
|:---------|:-----|
| textures | 纹理资源 | 
| i18n     | 国际化  |

#### 开发

- idea 启动客户端时缺少文件, 使用idea 重新同步项目即可.

##### 模块说明

| 模块              | 说明                                      |
|:----------------|:----------------------------------------|
| mc-api          | mc源码引入, 由于协议原因, 需要自己用官方映射编译然后放到对应位置上    |
| mc-core         | 基于mc源码的扩展实现, 用于方便进行mod开发                |
| mc-impl         | mod 本体实现                                |
| component-core  | 供所有模块使用的核心类                             |
| component-libs  | 所有模块依赖的三方类, 用于 idea启动和打包                |
| component       | 三方组件,  可视情况进行引入依赖或者被依赖                  |
| launch-core     | 用于适配多个mod启动方式的类                         |
| launch-neoforge | 专用于 neoforge 启动, 打包后的jar用于分发给neoforge平台 |
